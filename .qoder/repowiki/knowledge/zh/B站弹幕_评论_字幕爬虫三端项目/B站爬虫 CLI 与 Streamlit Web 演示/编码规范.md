- 所有对外抓取函数统一采用 `(bvid, max_age, credential, save_fmt, ...)` 签名，`max_age=0` 表示禁用缓存，非零值以秒为单位写入 `.bili_cache/*.json`。
- 保存文件名遵循 `{danmaku|comments|subtitle}_{bvid}[_{lan_code}].{ext}` 命名约定，便于 UI 层用 `Path.glob` 批量发现产物。
- CLI 入口使用 `argparse.RawTextFormatter` 并在 `epilog` 中嵌入示例命令，保持帮助文本的可读性。
- Streamlit 页面通过自定义 `CaptureIO` 类捕获底层 `print` 输出，再用 `st.empty().code()` 实时追加显示最后 50 行日志。
